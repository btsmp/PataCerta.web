# PataCerta

Bem-vindo ao **PataCerta** - uma plataforma dedicada à adoção de pets! Nosso objetivo é conectar pessoas apaixonadas por animais com seus futuros melhores amigos de quatro patas.

## Descrição do Projeto

O PataCerta é uma plataforma de adoção de pets que inclui:

- **Cadastro de Pets Disponíveis**: Permite que os donos cadastrem seus pets disponíveis para adoção.
- **Cadastro de Usuários**: Permite que os usuários se registrem e criem perfis.
- **Chat em Tempo Real**: Funcionalidade para comunicação entre adotantes e donos dos pets.
- **Requisição de Adoção**: Usuários podem manifestar interesse na adoção de um pet.

## Tecnologias Usadas

- **Frontend**: React, Next.js, Tailwind CSS
- **Backend**: NestJS, Prisma, PostgreSQL
- **Autenticação**: JWT, Cookies
- **Testes**: Storybook, Cypress
- **Hospedagem**: Vercel (Frontend), Hostinger (Backend)

## Instalação

Para iniciar o projeto, você precisará configurar o ambiente de desenvolvimento. Siga os passos abaixo para configurar o frontend e o backend.

### Frontend

1. Clone o repositório:

   ```bash
   git clone git@github.com:seu-usuario/pata-certa.git
   cd pata-certa
   ```
2. Instale as dependências:
    ```bash 
    npm install
   ```
